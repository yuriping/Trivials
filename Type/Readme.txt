typelen_longlongint
0. long int 类型是32位，long long int 类型是64位；
1. 立即数是32位的，使用表达式 1 << 40，会左移溢出，编译出现 “warning: left shift count >= width of type”警告；
   使用表达式 1LL << 40 指定立即数分配的内存为64位，可解决；
2. 一般来说，两个32位变量相乘，得到64位结果，但一个32位变量与立即数相乘，得到的仍然是32位结果，可以通过将32位变量强制类型转换为64位，或者，将立即数说明为64位，来得到64位积结果；
   但将一个32位变量与一个说明为32位（L）的立即数相乘，得到的依然是32位结果，机理不详……
